# Contributing

Hafiz is not accepting outside code contributions or unsolicited pull requests yet.

Issues are welcome:

- Bug reports
- Content corrections
- Accessibility problems
- Feature requests
- Production-readiness or documentation issues

Open issues at https://github.com/71iq/Hafiz/issues.

Unsolicited pull requests may be closed without review. If a maintainer asks you to open a pull request for a specific issue, keep it narrowly scoped to that request.
