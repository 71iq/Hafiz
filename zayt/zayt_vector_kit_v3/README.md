# Zayt Vector Kit v3

A richer, cuter, more 3D-feeling vector version of Zayt for Hafiz.

## Files

- `zayt_master_layered_v3.svg` — all layers in one SVG.
- `zayt_preview_v3.png` — raster preview on a Hafiz warm surface.
- `parts/zayt_glow.svg`
- `parts/zayt_arms.svg`
- `parts/zayt_body.svg`
- `parts/zayt_eyes.svg`
- `parts/zayt_mouth.svg`
- `parts/zayt_leaf.svg`

All SVGs use the same `viewBox="0 0 512 512"`, so parts can be stacked directly.

## Suggested animation usage

- **Again:** eyes soften/downcast, mouth switches to a tiny frown variant, arms move inward, body dips 2–3px.
- **Hard:** eyes close, body scale breathes 0.985 → 1.0, glow dims slightly.
- **Good:** glow opacity rises, mouth smile increases slightly, cheeks fade in.
- **Easy:** leaf does a tiny tilt, shimmer/glow pulses once.

## Design guardrails

Zayt is a study companion, not decoration for Quran text. Keep it outside Mushaf reading surfaces or use it only as a very minimal contextual guide.
